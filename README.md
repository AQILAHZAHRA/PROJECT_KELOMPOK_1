Sistem Informasi Bank Sampah BSU Mekar Swadaya
Deskripsi Proyek
Sistem Informasi Bank Sampah BSU Mekar Swadaya merupakan aplikasi berbasis web yang dikembangkan untuk membantu pengelolaan data dan transaksi pada Bank Sampah. Sistem ini dirancang untuk mempermudah proses pencatatan setoran sampah, pengelolaan data nasabah, perhitungan saldo secara otomatis, serta pembuatan laporan keuangan yang terstruktur dan akurat.
Aplikasi ini ditujukan untuk mendukung aktivitas operasional pengelola Bank Sampah agar lebih efisien, mengurangi kesalahan pencatatan manual, serta meningkatkan transparansi data.

Fitur Utama
Autentikasi pengguna (Login & Logout)
Manajemen data nasabah (Tambah, Ubah, Hapus)
Manajemen data jenis sampah dan harga per kilogram
Pencatatan transaksi setoran sampah
Perhitungan saldo nasabah secara otomatis
Riwayat transaksi nasabah
Pembuatan dan ekspor laporan transaksi (PDF)
Manajemen pengguna dan hak akses

Teknologi yang Digunakan
Bahasa Pemrograman: PHP
Framework: Laravel
Database: MySQL
Web Server: Apache (XAMPP)
Frontend: HTML, CSS, JavaScript, Bootstrap

Tools Pendukung:
Visual Studio Code
Git & GitHub
Postman (API Testing)
